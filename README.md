# 🚀 AI Career Navigator

An AI-powered career prediction system that analyzes a student's skills, academic performance (CGPA), and personal interests to suggest the most suitable tech career path, along with a detailed **6-month learning roadmap**.

## 🌟 Features
- **Smart Prediction:** Uses a Machine Learning model (Random Forest/Decision Tree) to predict roles.
- **Dynamic Roadmaps:** Generates a month-by-month study plan for 8+ tech roles.
- **Modern UI:** Glassmorphic dashboard built with HTML5, CSS3, and JavaScript.
- **FastAPI Backend:** High-performance Python API for seamless data processing.

## 🛠️ Tech Stack
- **Backend:** Python, FastAPI, Uvicorn
- **Machine Learning:** Scikit-learn, Pandas, Pickle
- **Frontend:** HTML, Tailwind-style CSS, Vanilla JavaScript

## 📂 Project Structure
- `main.py`: FastAPI server and prediction logic.
- `best_model.pkl`: The trained Machine Learning model.
- `index.html`: The interactive user interface.
- `README.md`: Project documentation.
